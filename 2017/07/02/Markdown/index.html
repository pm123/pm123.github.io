Markdown

发表于 2017-07-02 | 分类于 blog
第一次写博客，先学习一下Markdown, 这篇文章是我对学习markdown的使用的简单总结

前言

HTML是一种发布的格式，Markdown是一种书写的格式。可以直接在Markdown中混入HTML的内容，但是有以下几点需要注意：

  • 一些 HTML 块元素――比如 div、table、pre、p 等标签，必须在前后加上空行与其它内容区隔开，
    还要求它们的开始标签与结尾标签不能用制表符或空格来缩进。Markdown 的生成器有足够智能，
    不会在 HTML 区块标签外加上不必要的 p 标签。
  • HTML 块元素标签内的 Markdown 格式语法不会被处理。
  • HTML 行内元素可以在 Markdown 格式内随意使用

读Markdown使用文档中有两个概念要强调一下，区块元素和区段元素，区块元素对应 HTML 中的块元素，区段元素对应 HTML
中的行内元素（仅是我的理解）

区块元素

  • 段落
    • 前后要有一个以上的空行
    • 普通段落不要用空格或制表符缩进
  • 换行

  • 标题

    • 两种格式

    • 1 ~ 6 个 # 对应6级标题

      this is head     this is head
============     -----------
#
##
###

  • 引用

    • >表示引用
  • 列表
    • 无序列表在每一项开头加上“*”、“+”、“－”(减号好像不起作用)
    • 有序列表有每一项开关加上“1”、“2” …..
    • 列表项目标记通常是放在最左边，但是其实也可以缩进，最多 3 个空格，项目标记后面则一定要接着至少一个空格或制表符。

  • 代码区块

    • 只要简单地缩进 4 个空格或是 1 个制表符就可以
    • 一个代码区块会一直持续到没有缩进的那一行（或是文件结尾）。

    • 在代码区块里面， & 、 < 和 > 会自动转成 HTML 实体

      var arr = [];
console.log(arr);

<div class="footer">
     &copy; 2004 Foo Corporation
</div>

  • 分隔线

    • 你可以在一行中用三个以上的星号、减号、底线来建立一个分隔线，行内不能有其他东西。你也可以在星号或是减号中间插入空格
      * * *
***
- - -
_ _ _ _ _

区段元素

  • 链接

    • This is an example inline link.

      This is [an example](http://example.com/ "Title")

    • This link has no title attribute.

      [This link](http://example.net/)

    • example

      <a href="http://example.com/" title="Title">example</a>

  • 强调

    • Markdown 使用星号（）和底线（_）作为标记强调字词的符号，被 或 包围的字词会被转成用 em 标签包围，用两个 * 或 包起来的话，则会被转成 strong
    • 如果你的 * 和 _ 两边都有空白的话，它们就只会被当成普通的符号。
  • 代码
    • 如果要标记一小段行内代码，你可以用反引号把它包起来（`）
    • 在代码区段内，& 和方括号都会被自动地转成 HTML 实体，这使得插入 HTML 原始码变得很容易
      • Use the printf() function.
      • A single backtick in a code span: `
      • A backtick-delimited string in a code span: `foo`
  • 图片
    • 格式： ![]()
    • 到目前为止， Markdown 还没有办法指定图片的宽高，如果需要的话，可以使用普通的 标签。
